## Deploy on Render.com (FREE - 3 Steps Only)

### Step 1: Upload to GitHub
1. Go to github.com → New repository → Name it `fitpulse`
2. Upload ALL files from this zip (drag & drop)
3. Click "Commit changes"

### Step 2: Connect to Render
1. Go to render.com → Sign up (free)
2. Click "New +" → "Web Service"
3. Connect your GitHub → Select `fitpulse` repo

### Step 3: Deploy Settings
Fill these fields EXACTLY:
- **Language:** Node
- **Build Command:** `npm install`
- **Start Command:** `node server.js`
- **Instance Type:** Free

Click "Create Web Service" → Wait 2-3 minutes → Your URL is ready! ✅

---

## Deploy on Railway.app (Alternative)

1. Go to railway.app → Login with GitHub
2. "New Project" → "Deploy from GitHub repo"
3. Select `fitpulse` repo
4. Railway auto-detects Node.js — just click Deploy
5. Go to Settings → Domains → "Generate Domain"

## Run Locally (Your Computer)

Requirements: Node.js installed (nodejs.org)

```
npm install
node server.js
```

Open browser: http://localhost:3000
